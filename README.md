## ✨ About Rook Editor

**Rook Editor** is a lightweight and easy-to-use browser code playground where you can write **HTML, CSS, and JavaScript** and instantly see the output.

It is designed for **speed, simplicity, and learning**, making it perfect for developers who want a quick environment to test front-end code without installing heavy IDEs.

Whether you're **learning web development**, **testing UI ideas**, or **debugging snippets**, Rook Editor provides a clean and distraction-free workspace.

---
🌐 Live Demo

Try the editor directly in your browser:

👉 https://rookeditor.netlify.app/

No installation required.

## 🚀 Features

✔ **Realtime Output**
Instantly see your code results as you type.

✔ **Separate Editors**

* HTML Editor
* CSS Editor
* JavaScript Editor

✔ **Fast & Lightweight**
No installation. No frameworks. Just open and code.

✔ **Clean Interface**
Minimal and distraction-free coding environment.

✔ **Beginner Friendly**
Great for learning front-end development.

✔ **Runs Anywhere**
Works directly in any modern web browser.
